Migdal/, BR/ and NuclearRecoil/ contains the signal templates
for N_{e^-} >= 4 and for the following mass points:

Migdal @ m_{\chi} = 1 GeV,  \sigma_{SI} = 1e-37
BR     @ m_{\chi} = 1 GeV,  \sigma_{SI} = 1e-33
NR     @ m_{\chi} = 2 GeV,  \sigma_{SI} = 1e-41

The code used to compute these templates can be found at
https://github.com/ggrillidc/DDrates
